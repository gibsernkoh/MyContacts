# My Contact

This is an app for storing users' contact book.
New user will need to create an account.

## Stack

This project is done using Nuxt3 for the FE (View) and BE(Controller) with Planetscale (Serverless Database)

### How to start

```bash

# npm
npm i
npm run dev

```
